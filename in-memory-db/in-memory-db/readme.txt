Dieses kleine Beispiel zeigt wie man die im JDK enthaltene In-Memory Derby DB verwenden kann.
Zur Installation der DB ist nur ein JDK zu installieren und die Umgebungsvariablen zu setzen.

In einem "echten" Projekt muss man sich noch überlegen wie man zwischen einer vorhandenen
Datenbank und der In-Memory-DB wechselt.

Aus meiner Sicht gilt hier das gleiche wie für andere Ressourcen oder Abhängigkeiten.

- Interfaces
- Injection
